# Things-Manager
A CLI app to manage things like Book, Music, Movie etc.

## Built With

- Ruby

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Git
- Code Editor. e.g., VS Code
- Ruby

### Setup

- Clone the repository using the `git clone` command.

### Usage

- ruby index.rb
